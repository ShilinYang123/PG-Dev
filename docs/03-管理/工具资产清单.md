## 使用建议

1. **新项目启动**: 使用 `init_project.py` 进行初始化
2. **日常开发**: 使用 `utils.py` 中的通用函数
3. **结构检查**: 定期运行 `check_structure.py` 验证项目结构
4. **配置管理**: 通过 `config_loader.py` 和 `config_validator.py` 管理配置
5. **性能监控**: 使用 `performance_monitor.py` 监控系统性能
6. **测试验证**: 运行 `tests/` 目录下的测试确保代码质量
7. **项目完成**: 使用 `finish.py` 进行项目收尾工作
